# Análise Empírica de Algoritmos

Este projeto tem como objetivo analisar empiricamente o tempo de execução de algoritmos clássicos.

## Algoritmos analisados

* Busca Sequencial
* Busca Binária
* Insertion Sort

## Metodologia

Foram realizados testes com diferentes tamanhos de entrada, medindo o tempo médio de execução e comparando com funções assintóticas teóricas.

## Resultados

Os resultados confirmaram:

* Busca Sequencial → O(n)
* Busca Binária → O(log n)
* Insertion Sort → O(n²)
